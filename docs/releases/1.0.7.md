# SOMCP 1.0.7

本发布说明仅包含 `v1.0.6..v1.0.7` 的增量变化。

## 更新内容

- 审查并合并 PR #3，强化 SO/ELF 与 APK 工具的正确路由提示。
- 系统拆分 UI、MCP、工具目录和原生分析引擎中的超大职责，`NativeSoEngine` 收敛为 80 行兼容 facade。
- MCP 批处理模板与执行器独立，补充嵌套 JSONPath、失败中止和事务回滚测试。
- ASM 与符号批量编辑改为完整验证后一次提交，失败不再留下部分修改。
- APK 分析增加输入、条目数、单条目解压量和累计解压量限制，并修复标准 DEX magic 识别。
- LIEF JNI 增加异常屏障；LIEF 编辑仅在字节变化和后置条件成立时创建会话。
- MCP 初始化响应改为报告真实应用版本，README 与工具数量同步到当前实现。

## 验证

- 完整 JVM 单元测试。
- 四 ABI Release 原生编译。
- R8 压缩、资源优化和 Release 签名打包。
- Release APK 签名与 SHA-256 校验。
